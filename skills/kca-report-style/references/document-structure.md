# 표준 문서 골격

대부분의 KCA 보고서는 아래 4단 골격을 따른다. 섹션 표제는 문서 성격에 따라 유연하게 조정.

```
<제목>                         ← 18~20pt, 굵게, 가운데
< 날짜, 부서 >                  ← 12~13pt, 발신 정보 (예: < '26. 7. 3.(금), AI디지털심화팀 >)

□ 추진 배경                     ← 왜 하는가 (환경변화·필요성)
  ㅇ …
  ㅇ …

□ 추진 내용 (또는 "주요 내용")   ← 무엇을/어떻게 (핵심)
  ㅇ (추진 방향) …
  ㅇ (세부 과제) …
    - …
  [ 표 ] (구성안·커리큘럼·지표 등, 선택)

□ 향후 계획 (또는 "추진 일정")   ← 언제/다음 단계
  ㅇ … : '26. 6월 중
  ㅇ … : '26. 6월 ~ 7월 중
  ※ …
```

## 섹션 표제 변형

| 표준 | 대체 표현 |
|---|---|
| 추진 배경 | 추진 배경, 배경 및 필요성, 현황 및 문제점 |
| 추진 내용 | 주요 내용, 세부 추진계획, 추진 내용 |
| 향후 계획 | 추진 일정, 향후 계획, 기대 효과 및 일정 |

## 대규모 전략보고서 골격 (선택)

전략·계획 종합보고서는 아래 확장 골격 사용 가능:

```
Ⅰ. 성과 및 환경 분석 (성과·동향·SWOT)
Ⅱ. 전략 체계 (전략체계도·전략목표·과제)
Ⅲ. 세부 추진계획 (과제별 < 추진방향 > 박스 + 개조식)
Ⅳ. 모니터링 및 환류
```

- 각 세부과제는 `◇ < 추진방향 >` 박스 → `ㅇ`/`-` 개조식 전개.
- 전략목표는 로마자 `Ⅰ.Ⅱ.Ⅲ.`, 세부실행과제는 `①②③…` 원문자.

## 발신 정보 규칙

- 형식: `< 날짜(요일), [본부] 부서명 >`
- 예: `< '26. 6. 8.(월), 경영기획본부 AI디지털심화팀 >`
