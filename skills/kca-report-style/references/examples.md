# 실제 코퍼스 예시 (gold standard)

한국방송통신전파진흥원 AX 전략(2026) 및 교육계획(안) 원문 발췌. 재현 목표.

## 예시 1 — 추진 배경 (교육계획안)

```
□ 추진 배경

ㅇ AI 중심의 업무 환경 변화와 정부·공공기관의 AI 활용 확산 흐름에 따라, 전 직원의 AI·데이터 역량 강화 필요성 지속 증대

ㅇ AI·데이터 역량·교육 수요조사* 결과에 따라 교육 정례화·단계별 교육 확대 및 AI 윤리 등을 반영한 체계적 교육 운영 필요

* 조사기간: '26.5.8.~5.22. / 참여인원: 전 직원 중 242명 (57%) 응답
```

관찰: `ㅇ` 각 항목 2줄 이내·명사형("지속 증대", "운영 필요"). 나열 가운뎃점("정부·공공기관", "정례화·단계별"). 각주 `*` 하단 정의.

## 예시 2 — 주요 내용 + 괄호 리드 (전략보고서)

```
ㅇ (일상화 전략 수립) 기관 내 AI 일상화 추진을 위한 'AI를 가장 잘 활용하는 공공기관 실현' 전략 수립

ㅇ (AX 협의체 조직) CAIO·BAIO 임명을 통해 전략 실행력·사업별 오너십 강화 및 행정·업무 전반에 체계적 AX 내재화 추진

- 기관 AI 추진 과제 검토, 공모전 심사 등 국민 의견 수렴·반영
```

관찰: 괄호 리드 소제목 + 명사형 마감("전략 수립", "내재화 추진"). `-` 세부는 근거·부연.

## 예시 3 — < 추진방향 > 박스 + 세부과제 (전략보고서)

```
◇ 외부 통신이 차단된 온프레미스 환경 구축으로 데이터 보안 및 기술 주권 확보
◇ 정보시스템과 연동된 AI Agent 도입으로 복잡한 업무 프로세스 자동화

○ (GPU서버 도입) 내부망 전용 고성능 GPU(H200×2ea) 서버 도입을 통한 대규모 연산 가능한 독립적 AI 구동 환경 마련

- 외부 API 연동 없이 내부 데이터 처리 가능한 온프레미스 환경 구축으로 정보 유출 원천 차단
```

## 예시 4 — 향후 계획 / 추진 일정

```
□ 추진 일정

ㅇ AI활용 초급 1차 교육 구성 및 실시 : '26. 6월 중

ㅇ 1차 파일럿 프로젝트 참여자 수요조사 및 구성 : '26. 6월 중

ㅇ AI활용 중급 교육 대상 수요조사 및 교육 실시 : '26. 6월 ~ 7월 중

※ 단계별 교육 후 만족도 조사 및 참여율 향상을 위한 방안 추진
```

관찰: 일정은 `항목 : 시기` 형식. `※`로 후속 단서.

## Before / After — 초안 변환 예시

**Before (줄글 초안):**
> 우리 기관은 AI 활용이 확산되는 흐름에 맞춰 직원들의 역량을 강화할 필요가 있습니다. 그래서 초급, 중급, 고급으로 나눈 3단계 교육을 운영하려고 합니다. 초급은 분기별로 진행하고 AI 윤리와 동향을 다룹니다.

**After (개조식):**
```
□ 추진 배경
ㅇ AI 활용 확산에 따른 전 직원 역량 강화 필요성 증대

□ 추진 내용
ㅇ (추진 방향) 초급·중급·고급 3단계 교육 과정 운영
  - 초급: 분기별 운영, AI 윤리·동향 중심
```

변환 규칙: 서술어 명사화, "그래서/그리고" 등 연결어 제거, 나열 가운뎃점, 골격 배치.
