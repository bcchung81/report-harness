# 개조식 스타일 규칙 (상세)

한국방송통신전파진흥원 AX 전략 보고서 및 교육계획(안)에서 추출한 문체 규칙.

## 1. 계층 기호 체계

```
□ 섹션/대주제 표제 (주체·주제형, 1줄)
  ㅇ (소제목) 항목 설명 — 괄호 리드 후 명사형 마감 (2줄 이내)
    - 세부 부연·근거 (2줄 이내)
    ※ 부연·단서 사항 (항목 아래, 계층 무관)
    * 각주형 용어·수치 주석 → 본문에 * 표시, 블록 하단에 정의
  ㅇ ...
◇ < 추진방향 > 박스 안 요지 (계획 단위 상단)
①②③ 세부 실행과제·단계 열거
```

- 들여쓰기로 계층을 시각화한다(기호 자체가 계층을 나타내되 하위는 들여쓴다).
- `ㅇ`은 문서에 따라 `○`(속 빈 원)으로도 표기 — 동일 계층.

## 2. 명사형 종결 (가장 중요)

모든 항목은 **체언(명사)으로 마감**한다. 서술어를 명사화한다.

| 서술형(❌) | 명사형(✅) |
|---|---|
| 역량을 강화합니다 | 역량 강화 |
| 필요성이 증대되고 있다 | 필요성 지속 증대 |
| 교육을 운영한다 | 교육 운영 |
| 확보하기 위해 추진 | 확보를 위한 추진 |
| 도입할 예정이다 | 도입 추진 / 도입 예정 |

금지 어미: `-다, -합니다, -이다, -했다, -된다, -하였음`(단 `-임/-음` 명사화는 문맥상 허용하되 남용 금지).

## 3. 괄호 리드 패턴

`ㅇ` 항목은 앞에 **괄호로 소제목**을 세우고 설명을 잇는다.

```
ㅇ (추진 방향) AI·데이터 이해도 및 활용능력 강화를 위한 3단계 과정 운영
ㅇ (모델 전략) 업무 난이도·유형에 따른 오픈소스·상용 모델 복합 멀티 모델 전략 수립
```

- 괄호 안은 4–8자 핵심어. 뒤 설명은 명사형 마감.

## 4. 압축·전문용어

- 한자어·전문용어 중심으로 압축한다("정립", "내재화", "고도화", "확산", "선제적").
- 영문 약어·병기 허용: `RAG`, `PoC`, `ROI`, `On-Premise(온프레미스)`, `sLLM`.
- 불필요한 조사·연결어("그리고", "또한", "~에 대하여")를 절제한다.

## 5. 약물(약물기호) 사용

| 기호 | 의미 | 예 |
|---|---|---|
| `※` | 부연·단서·추가정보 | ※ 전직원 설문조사('26.1월)를 통해 수요분석 |
| `*` | 각주 주석(용어·수치 정의) | 본문 `수요조사*` → 하단 `* 조사기간: …` |
| `⇰`/`⇨` | 귀결·도출 | [성과]…[환경]… ⇰ 자주적·실용적 AX 혁신 |
| `·` | 나열(조사 대체) | 산업·공공, 전파·방송·기금 |
| `∙` | 표 내부 불릿 | ∙ 기관 사업별 업무특화 도구 제작 |

## 6. 날짜·수치 표기

- 연도 축약: `'26`, `'25.9`, `'26.5.8.~5.22.`
- 요일 병기: `'26. 6. 8.(월)`
- 수치·비율: `242명 (57%)`, `H200×2ea`, `7B~14B`, `120B`

## 7. 표 활용

- 표 앞에 `[ 제목(안) ]` 캡션(13pt)을 둔다.
- GFM 표 문법으로 작성. 표 내부는 짧은 명사구·`∙` 불릿.
- 커리큘럼·구성·성과지표 등 3열 구조 자주 사용(구분 | 목표/비고 | 내용).
